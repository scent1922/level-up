# Level-Up: 습관 관리 × 쉘터 육성 게이미피케이션 앱

## Product Requirements Document (PRD)

**버전**: 1.0 (MVP)
**작성일**: 2026-03-24
**프로젝트명**: Level-Up
**GitHub**: https://github.com/scent1922/level-up

---

## 1. 제품 개요

### 1.1 한 줄 요약
일상 습관을 퀘스트로 관리하고, 달성 보상으로 포스트아포칼립스 지하 쉘터를 육성·꾸미는 게이미피케이션 모바일 앱.

### 1.2 핵심 가치 제안 (Value Proposition)
| 대상 | 가치 |
|------|------|
| 습관을 만들고 싶지만 동기부여가 부족한 사람 | 쉘터 육성이라는 감정적 연결고리로 지속 동기 제공 |
| 기존 습관 앱이 지루한 사람 | RPG식 레벨/XP/리워드로 성취감 극대화 |
| 게임은 좋아하지만 시간 낭비 죄책감이 있는 사람 | 습관 실천 = 게임 진행, 생산적 게이미피케이션 |

### 1.3 핵심 컨셉
- **퀘스트 시스템**: 유저가 직접 습관을 퀘스트로 생성·관리
- **레벨 & XP**: 퀘스트 완료 시 경험치 획득, 레벨업으로 쉘터 확장/기능 추가
- **포인트 리워드**: 특정 조건 달성 시 포인트 적립, 쉘터/아바타 꾸미기에 사용
- **쉘터 육성**: 포스트아포칼립스 1인용 지하 벙커를 2D 픽셀아트 아이소메트릭 뷰로 육성
- **손실 회피**: 2주 방치 시 쉘터 파괴 → 강력한 지속 동기

## 2. 타겟 유저

### 2.1 주요 타겟
- **연령**: 18~35세
- **특성**: 자기 개발에 관심이 있고, 게임(특히 인디/픽셀아트/시뮬레이션)을 즐기는 MZ세대
- **사용 맥락**: 아침 루틴, 운동, 공부, 물 마시기 등 반복 습관을 형성하고자 하는 상황

### 2.2 유저 페르소나
> **김민준 (26세, 직장인)**
> 매일 아침 운동하겠다고 다짐하지만 3일을 못 넘김. 습관 앱을 깔아봤지만 체크리스트만 있으니 재미가 없어서 곧 삭제함. 학생 때 다마고치와 심시티에 빠졌던 경험이 있음. "내가 키운 무언가가 망가지는 건 못 참는다."

## 3. 기능 명세 (MVP Scope)

### 3.1 온보딩

| ID | 기능 | 상세 |
|----|------|------|
| ONB-01 | 쉘터 선택 | 앱 최초 실행 시 4개 쉘터 프리셋 중 1개 선택 |
| ONB-02 | 아바타 선택 | 4개 아바타 프리셋 중 1개 선택 |
| ONB-03 | 첫 퀘스트 생성 가이드 | 온보딩 플로우에서 최소 1개 퀘스트 생성 유도 |

**쉘터 프리셋 (MVP 4종)**:
1. **도시 벙커** — 폐허가 된 도시 지하, 콘크리트 벽, 비상등
2. **해안 벙커** — 절벽 아래 해안 동굴 기반, 파이프와 방수벽
3. **산림 벙커** — 산 속 지하, 나무 뿌리가 천장을 관통, 자연광 유입
4. **사막 벙커** — 오아시스 인근 지하, 모래색 벽, 냉각 장치

### 3.3 레벨 & XP 시스템

| ID | 기능 | 상세 |
|----|------|------|
| LVL-01 | XP 획득 | 퀘스트 1회 완료 시 기본 XP 획득 |
| LVL-02 | 레벨업 | 누적 XP가 레벨 요구치 도달 시 레벨업 |
| LVL-03 | 레벨업 보상 | 레벨업 시 포인트 보상 + 쉘터 업그레이드 잠금해제 |
| LVL-04 | 연속 달성 보너스 | 연속 N일 퀘스트 완료 시 XP 배율 증가 (스트릭) |
| LVL-05 | 레벨 표시 | 쉘터 화면 및 프로필에 현재 레벨/XP 바 표시 |

**XP 밸런싱 (초안)**:

| 레벨 구간 | 필요 누적 XP | 레벨당 증가분 | 체감 난이도 |
|-----------|-------------|--------------|------------|
| 1~10 | 100 / 레벨 | +100 | 빠른 성장기 (약 1~2주) |
| 11~25 | 200 / 레벨 | +200 | 안정기 (약 1~2개월) |
| 26~50 | 350 / 레벨 | +350 | 중급자 구간 |
| 51+ | 500 / 레벨 | +500 | 장기 유지 구간 |

**XP 획득량**:
- 퀘스트 1회 완료: 20 XP (기본)
- 하루 전체 퀘스트 완료: 추가 30 XP (데일리 보너스)
- 스트릭 보너스: 7일 연속 → 1.5배, 30일 연속 → 2배

### 3.6 쉘터 파괴 메커니즘

| ID | 기능 | 상세 |
|----|------|------|
| DTH-01 | 방치 카운터 | 마지막 퀘스트 완료일로부터 경과일 추적 |
| DTH-02 | 경고 단계 | 7일 방치: 쉘터 비주얼 열화 시작 (먼지, 어두워짐) |
| DTH-03 | 위험 알림 | 10일 방치: 푸시 알림 "쉘터가 위험합니다" + 비주얼 심각 열화 |
| DTH-04 | 쉘터 파괴 | 14일 방치: 쉘터 파괴 연출 → 아바타 사망 |
| DTH-05 | 재시작 | 파괴 후 새 쉘터/아바타 프리셋 재선택, 레벨 1부터 시작 |
| DTH-06 | 구제 시스템 | 파괴 직전 1회 부활 기회 (포인트 대량 소비) |

**DTH-06 구제 시스템 설계 의도**: 2주 방치 → 즉시 파괴는 복귀 의지를 완전히 꺾을 수 있음. "포인트를 대량 소비해서라도 살린다"는 선택지가 있으면 복귀 유인이 생김. 단, 무한 부활은 불가 — 연속 부활 시 비용 급증 또는 횟수 제한.

### 4.3 에셋 전략

- **사전 제작**: 모든 쉘터/아바타/오브젝트 에셋을 AI(이미지 생성)로 사전 제작
- **번들링**: 앱 바이너리에 포함 (CDN 불필요, 오프라인 동작)
- **스프라이트 시트**: 아바타 모션은 스프라이트 시트로 제작 (idle, walk, interact 등)
- **스타일 가이드**: 통일된 픽셀아트 스타일 (32×32 또는 64×64 타일 기준, 제한 팔레트)

**MVP 에셋 목록**:

| 카테고리 | 수량 | 내용 |
|---------|------|------|
| 쉘터 베이스 | 4종 | 도시/해안/산림/사막, 각 3단계 확장 |

## 10. 구현 순서 (MVP 마일스톤)

| **M1: 프로젝트 셋업** | Expo 초기화, 네비게이션, DB 스키마, 타입 정의 | 10% |
| **M2: 퀘스트 시스템** | CRUD, 반복 주기, 완료 체크, 목록/캘린더 뷰 | 25% |

| **M7: 알림 & 방치 시스템** | 푸시 알림, 방치 감지, 열화/파괴 연출 | 5% |
